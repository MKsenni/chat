# Chat

SPA-чат для общения внутри рабочих групп. Экран с историей сообщений, ботом-собеседником и оптимистичными обновлениями.

## Задачи

- Список сообщений с аватаром, именем, текстом и временем
- Отправка сообщений через поле ввода (Enter / кнопка)
- Бот отвечает случайной фразой с задержкой 1–2 сек
- Статусы сообщений: sending → sent → read / failed + Retry
- Оптимистичное обновление — сообщение появляется в списке до подтверждения сервера
- Эмуляция сбоя отправки (~20%) с возможностью повторной отправки
- Персистентность истории через localStorage
- Индикатор «печатает...»

## Стек

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Zustand (стейт + persist)
- React Compiler (babel-plugin-react-compiler)

## Запуск

